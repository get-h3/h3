The Path: How to Actually Use H3

1
Build a harness in Go, Python, or TypeScript that implements /v1/process and /v1/result.
hermes h3 scaffold --lang go generates the boilerplate.
2
Verify compliance with the 43-test battery.
h3-test --endpoint http://localhost:9191
3
Register the harness with Hermes so sessions can be routed through it.
hermes h3 install http://localhost:9191
4
Route a chat through H3. Every message now goes to your harness instead of Hermes' native agent loop.
hermes h3 route telegram:<chat_id>
5
Message that chat. Hermes builds the context (tools, models, history, memory, skills), calls your harness via POST /v1/process, executes the decision, feeds results back via POST /v1/result, and loops until your harness returns {"decision":"end"}.
